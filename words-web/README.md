# Words

Интервальное повторение слов

Есть пассивное изучение слов (чтение) и активное (письмо).

Цель пассивного изучения слов - узнавать слова в тексте.

Карточка состоит из
1. предолжение. Изучаемое слово заменено ___
2. делается подсказка на нидерландском языке
3. если совсем туго, то коротко на русском
4. в итоге полное предлоежине.

Лицевая сторона - это предложение, где я должен вспомнить (не прописать, а про себя) слово

Обратная стора - обясненние слова.

ChatGPT готоворит, что после нескольких повторов, я даже не буду смотреть описание на голланском, а сразу вспоминать конекст и пропущенное слово.

Цель активного изучения слов - правильно писать и строить предложения.

Карточка активного изучения слов состоит и поля ввода, куда надо ввести правильное слово.
